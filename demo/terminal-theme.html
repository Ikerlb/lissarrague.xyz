rust-ray-tracer

rust graphics rayon

a ray tracer implementation following "Ray Tracing in One Weekend" by Peter Shirley, written in Rust. uses Rayon for data parallelism to render scenes across all available CPU cores.

// the core render loop fn render(scene: &Scene, width: u32, height: u32) -> Image { (0..height).into_par_iter().map(|y| { (0..width).map(|x| { let ray = camera.get_ray(x, y); trace(&ray, scene, 0) }).collect() }).collect() }

the final scene renders ~500 spheres with diffuse, metal, and dielectric materials. on a modern CPU it finishes in about 8 seconds at 1200x800.
